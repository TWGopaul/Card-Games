Program to simulate electronic card games found at casinos.
Player can choose 5 card draw poker or black jack.

Poker.h - Header file for poker game logic
Poker.cpp - Member functions for poker.h

BlackJack.h - Header file for black jack game logic
BlackJack.cpp - Member functions for BlackJack.h

Card.h - Header file for Card class structure and Acard struct\
Card.cpp - Member functions for Card class

Project3.cpp - Driver program for the card games
